### ASSISTENTE

Felippe, pelo que entendi, você deseja utilizar seu SSD de 120GB para acelerar a inicialização do Windows, mas manter os aplicativos e arquivos no HD externo de 1TB para evitar problemas de espaço. Uma solução possível seria configurar o SSD como uma unidade de cache para o HD externo. Isso pode ser feito através de uma tecnologia chamada "Tiered Storage" no Windows, mas requer um pouco de conhecimento técnico avançado e pode não ser diretamente suportado no Windows 11.

No entanto, há uma alternativa mais simples e viável: mover a localização padrão de instalação de novos aplicativos para o HD externo e usar o SSD apenas para o sistema operacional. Aqui estão os passos para isso:

1. **Instalar o Windows no SSD**: Primeiro, instale o Windows 11 no SSD. Isso garantirá que o sistema operacional seja carregado rapidamente.

2. **Alterar a localização de instalação de novos aplicativos**: No Windows 11, você pode mudar a localização padrão de novos aplicativos. Vá em Configurações > Sistema > Armazenamento > Alterar onde o novo conteúdo é salvo. Aqui, você pode definir o novo local de instalação de aplicativos para o HD externo.

3. **Mover aplicativos existentes**: Para os aplicativos já instalados no SSD, você pode movê-los para o HD externo. Vá em Configurações > Aplicativos > Aplicativos e recursos, selecione um aplicativo e clique em "Mover" (se disponível) para transferi-lo para o HD extern
