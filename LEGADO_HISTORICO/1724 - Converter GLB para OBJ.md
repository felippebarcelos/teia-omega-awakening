# 1724 - Converter GLB para OBJ

### USUÁRIO

Qual app na appstore possibilita gratuitamente converter arquivos glb em obj

---

### ASSISTENTE

Atualmente, não há aplicativos gratuitos na App Store que permitam converter arquivos GLB para OBJ diretamente no iPhone ou iPad. No entanto, você pode utilizar ferramentas online gratuitas que funcionam perfeitamente em navegadores móveis, como o Safari no iOS, para realizar essa conversão.

### 🔄 Ferramentas Online Gratuitas para Converter GLB em OBJ

Essas plataformas permitem que você faça upload de arquivos `.glb` e os converta para `.obj` sem a necessidade de instalar aplicativos adicionais:

1. **AnyConv**  
   Conversor online gratuito que suporta arquivos de até 100 MB. Basta fazer o upload do arquivo GLB e baixar o OBJ convertido. citeturn0search1

2. **ImageToStl**  
   Permite a conversão de até 25 arquivos simultaneamente, com suporte a arquivos de até 100 MB. Ideal para conversões em lote. citeturn0search3

3. **3DPEA**  
   Oferece uma interface simples para conversão rápida de arquivos GLB para OBJ, sem necessidade de cadastro. citeturn0search6

4. **FabConvert**  
   Suporta upload de até 20 arquivos por vez, com tamanho máximo de 100 MB cada. Após a conversão, os arquivos são excluídos dos servidores após 15 minutos, garantindo maior privacidade. citeturn0search12

5. **MiConv**  
   Conversor online gratuito com limite de tamanho de arquivo de 75 MB. Os arquivos são excluídos automaticamente após 2 horas. citeturn0search11

### 📱 Aplicativo com Exportação para OBJ

Embora não seja exclusivamente um conversor, o aplicativo **Qlone** permite escanear objetos em 3D usando a câmera do iPhone e exportar os modelos nos formatos OBJ, GLB, FBX, entre outros. É uma opção útil se você estiver interessado em criar modelos 3D diretamente do seu dispositivo. citeturn0search24

Para converter arquivos GLB em OBJ diretamente no seu dispositivo iOS, recomenda-se o uso das ferramentas online mencionadas acima, acessíveis via navegador. Caso você precise criar modelos 3D a partir de objetos físicos, o Qlone pode ser uma solução prática.
