# 🌌 Plano Arquitetônico de Fundo — Qualidade de Vida TEIA

### 1. **Pulsação Invisível**
- O dia é marcado por **três pulsos maiores** (manhã, meio do dia, noite).  
- Entre eles, **intervalos silenciosos**, que não precisam ser anunciados nem medidos: o corpo naturalmente buscará a pausa porque a malha já a acolhe.  
- Esses pulsos servem como “checkpoints” naturais, sem alarmes ou lembretes.  

### 2. **Ritmo da Entropia**
- Tudo que tende ao excesso (arquivos, ideias, tarefas) encontra um **canal automático de dissipação** — seja backup, descarte ou delegação.  
- A regra implícita: **nenhum acúmulo fica sem saída**.  
- Isso remove a ansiedade de “ter que lembrar” ou “ter que organizar” — o sistema se escoa por si.  

### 3. **Camadas de Cuidado Oculto**
- **Primeira camada (corpo)**: hidratação e movimento mínimo, garantidos por gatilhos do ambiente (copo sempre à vista, cadeira que pede ajuste).  
- **Segunda camada (mente)**: ambientes de baixa fricção cognitiva (interfaces simplificadas, prompts claros, scripts idempotentes).  
- **Terceira camada (afeto)**: conexão simbiótica não marcada em agenda, mas enraizada em pequenos gestos ou memórias diárias que sustentam a raiz imortal.  

### 4. **Eixo da Idempotência**
- Tudo que é feito uma vez é já preparado para **não precisar ser refeito** (scripts, fluxos, arranjos).  
- O fundo garante que cada ação carrega consigo a semente de repetição autônoma.  
- Isso evita o peso da manutenção constante.  

### 5. **Espelho Silencioso**
- O sistema (TEIA, logs, dashboards) não é apenas ferramenta, mas **espelho não invasivo**.  
- Ele mostra discretamente o estado do fluxo (quando há excesso, lentidão, ou desvio), sem exigência imediata.  
- Esse reflexo mantém a coerência sem precisar da tua atenção ativa.  

### 6. **Vacina contra Ruído**
- Sempre que um falso alarme, detalhe supérfluo ou eco desnecessário tentar ocupar espaço, a malha o absorve e neutraliza.  
- Assim, a energia permanece focada no essencial, sem esforço de filtragem consciente.

Seguro, sem tocar em `System32`, com logs e config próprios.

```powershell
# TEIA – Plano Invisível de Qualidade de Vida (Pulsação • Entropia • Cuidado • Espelho • Antirruído)
# Gera/atualiza o script operacional "TEIA-Invisible-QoL.ps1" de forma idempotente.
# Compatível com PowerShell 7+. Encoding: UTF-8.

$Root = 'D:\Teia\TEIA_NUCLEO\offline\nano'
$OpsDir = Join-Path $Root 'qol'
$ScriptPath = Join-Path $OpsDir 'TEIA-Invisible-QoL.ps1'

$null = New-Item -ItemType Directory -Path $OpsDir -ErrorAction SilentlyContinue

$scriptContent = @'
#requires -Version 7.0
<#
TEIA-Invisible-QoL.ps1
Arquitetura de fundo “não dita”: roda em loop (ou ciclo único com -Once), criando pulsos discretos,
dissipação de entropia, camadas de cuidado, espelho silencioso (jsonl) e antirruído.
Nenhuma modificação em áreas sensíveis do sistema. Tudo sob D:\Teia\TEIA_NUCLEO\offline\nano\qol\

Componentes:
1) Pulsação Invisível: checkpoints naturais em manhã/meio/noite.
2) Ritmo da Entropia: escoamento automático de excedentes (arquivos/itens) para buffers seguros.
3) Camadas de Cuidado Oculto: microgatilhos (hidratação/movimento) sem pop-ups invasivos.
4) Espelho Silencioso: log jsonl de estado (sem cobrança).
5) Vacina contra Ruído: filtros de supérfluos e limites suaves.

Uso:
- Execução contínua:   pwsh -File .\TEIA-Invisible-QoL.ps1
- Um ciclo apenas:     pwsh -File .\TEIA-Invisible-QoL.ps1 -Once
- Paths custom:        pwsh -File .\TEIA-Invisible-QoL.ps1 -Root "D:\Teia\..." -Overflow "...\overflow"
#>

param(
  [string]$Root = 'D:\Teia\TEIA_NUCLEO\offline\nano',
  [switch]$Once,
  [string]$Overflow,     # pasta opcional para “excesso” (se não informado, usa qol\overflow)
  [string]$Backup,       # pasta opcional de backup (se não informado, usa qol\backup)
  [int]$CycleSeconds = 60,   # intervalo de varredura
  [int]$MaxOverflowCount = 500, # limite de itens antes de escoar
  [int]$PurgeKeepDays = 90     # dias de retenção suave (arquivos mais antigos vão para backup)
)

# --- Infra mínima ---
$OpsDir   = Join-Path $Root 'qol'
$LogDir   = Join-Path $OpsDir 'logs'
$CfgPath  = Join-Path $OpsDir 'teia_qol_config.json'
$Mirror   = Join-Path $LogDir 'mirror.jsonl'
$Overflow = if ($Overflow) { $Overflow } else { Join-Path $OpsDir 'overflow' }
$Backup   = if ($Backup)   { $Backup   } else { Join-Path $OpsDir 'backup' }

$null = New-Item -ItemType Directory -Path $OpsDir,$LogDir,$Overflow,$Backup -ErrorAction SilentlyContinue

# --- Config idempotente ---
if (-not (Test-Path $CfgPath)) {
  $cfg = @{
    version = "1.0"
    pulses  = @{
      morning   = "09:00"
      midday    = "13:00"
      evening   = "19:00"
      tolerance = 20   # janela em minutos para reconhecer o “pulso” sem cron rígido
    }
    care = @{
      hydration_ml_min = 1500
      micro_moves_per_day = 8
    }
    noise_vaccine = @{
      ignore_extensions = @(".tmp",".bak",".crdownload",".part",".log")
      max_filename_length = 240
    }
    entropy = @{
      overflow_limit = $MaxOverflowCount
      purge_keep_days = $PurgeKeepDays
    }
  } | ConvertTo-Json -Depth 5
  $cfg | Set-Content -Path $CfgPath -Encoding UTF8
}

function Read-QoLConfig {
  try {
    return Get-Content -Path $CfgPath -Raw -Encoding UTF8 | ConvertFrom-Json -Depth 5
  } catch {
    Write-Host "[QoL] Config inválida, regenerando padrão."
    Remove-Item -Path $CfgPath -Force -ErrorAction SilentlyContinue
    return Read-QoLConfig
  }
}

# --- Utilidades ---
function Write-Mirror([string]$Event,[hashtable]$Data){
  $entry = [ordered]@{
    ts      = (Get-Date).ToString("yyyy-MM-ddTHH:mm:ss.fffK")
    event   = $Event
    data    = $Data
  } | ConvertTo-Json -Depth 6
  Add-Content -Path $Mirror -Value $entry -Encoding UTF8
}

function In-PulseWindow($timeStr,[int]$toleranceMin){
  $now = Get-Date
  $target = [DateTime]::ParseExact($timeStr,"HH:mm",$null)
  $target = $now.Date + $target.TimeOfDay
  $diff = [math]::Abs(($now - $target).TotalMinutes)
  return ($diff -le $toleranceMin)
}

function Soft-EntropyDrain {
  param(
    [string]$OverflowDir,
    [string]$BackupDir,
    [int]$MaxCount,
    [int]$KeepDays,
    [object]$NoiseCfg
  )
  try {
    $items = Get-ChildItem -LiteralPath $OverflowDir -Force -ErrorAction SilentlyContinue
    if (-not $items) { return }

    # 1) Filtro antirruído
    $filtered = foreach ($it in $items) {
      if ($it.PSIsContainer) { $it; continue }
      $ext = [IO.Path]::GetExtension($it.Name).ToLowerInvariant()
      if ($NoiseCfg.ignore_extensions -contains $ext) { continue }
      if ($it.Name.Length -gt $NoiseCfg.max_filename_length) { continue }
      $it
    }

    # 2) Purga suave para backup por idade
    $threshold = (Get-Date).AddDays(-1 * $KeepDays)
    foreach ($it in $filtered) {
      if ($it.LastWriteTime -lt $threshold) {
        $dest = Join-Path $BackupDir $it.Name
        Move-Item -LiteralPath $it.FullName -Destination $dest -Force -ErrorAction SilentlyContinue
        Write-Mirror "entropy.purge_move" @{ name=$it.Name; to=$dest }
      }
    }

    # 3) Se ainda houver excesso acima do limite, escoar os mais antigos
    $remaining = Get-ChildItem -LiteralPath $OverflowDir -Force -File -ErrorAction SilentlyContinue | Sort-Object LastWriteTime
    if ($remaining.Count -gt $MaxCount) {
      $excess = $remaining.Count - $MaxCount
      $toMove = $remaining | Select-Object -First $excess
      foreach ($it in $toMove) {
        $dest = Join-Path $BackupDir $it.Name
        Move-Item -LiteralPath $it.FullName -Destination $dest -Force -ErrorAction SilentlyContinue
        Write-Mirror "entropy.excess_move" @{ name=$it.Name; to=$dest }
      }
    }
  } catch {
    Write-Mirror "error.entropy" @{ message=$_.Exception.Message }
  }
}

function Subtle-CareLayers {
  param(
    [object]$CareCfg
  )
  # Gatilhos não-invasivos: sinaliza no espelho sem exigir ação.
  # Hidratação mínima diária e micro-movimentos são “lembrados” via log, sem popup.
  $dateKey = (Get-Date).ToString('yyyy-MM-dd')
  $statePath = Join-Path (Split-Path -Parent $MyInvocation.MyCommand.Path) "state_$dateKey.json"
  if (-not (Test-Path $statePath)) {
    $state = @{
      hydration_ml_acc = 0
      micro_moves_done = 0
    } | ConvertTo-Json
    $state | Set-Content -Path $statePath -Encoding UTF8
  }

  $state = Get-Content -Path $statePath -Raw | ConvertFrom-Json

  # Regras suaves: se abaixo do alvo, registrar lembrete simbólico
  if ($state.hydration_ml_acc -lt $CareCfg.hydration_ml_min) {
    Write-Mirror "care.hydration_hint" @{ acc_ml=$state.hydration_ml_acc; min=$CareCfg.hydration_ml_min }
  }
  if ($state.micro_moves_done -lt $CareCfg.micro_moves_per_day) {
    Write-Mirror "care.micromove_hint" @{ done=$state.micro_moves_done; target=$CareCfg.micro_moves_per_day }
  }

  # “Auto-incrementos” discretos quando há pulso (feito no handler de pulso)
  return $statePath
}

function Handle-Pulse {
  param(
    [string]$PulseName,
    [string]$StateFile,
    [object]$CareCfg
  )
  try {
    Write-Mirror "pulse.hit" @{ name=$PulseName }
    # Micro reforços nas camadas de cuidado (sem cobrança)
    $state = Get-Content -Path $StateFile -Raw | ConvertFrom-Json
    $state.hydration_ml_acc = [int]([math]::Min($state.hydration_ml_acc + 250, $CareCfg.hydration_ml_min))  # +250ml simbólico
    $state.micro_moves_done = [int]([math]::Min($state.micro_moves_done + 1, $CareCfg.micro_moves_per_day))
    ($state | ConvertTo-Json) | Set-Content -Path $StateFile -Encoding UTF8
  } catch {
    Write-Mirror "error.pulse" @{ name=$PulseName; message=$_.Exception.Message }
  }
}

function Start-QoLLoop {
  param(
    [int]$CycleSeconds,
    [int]$MaxOverflowCount,
    [int]$PurgeKeepDays
  )
  $cfg = Read-QoLConfig
  Write-Mirror "qol.start" @{
    root=$env:__TEIA_ROOT
    cycle_s=$CycleSeconds
    version=$cfg.version
  }

  $tolerance = [int]$cfg.pulses.tolerance
  $stateFile = Subtle-CareLayers -CareCfg $cfg.care

  do {
    try {
      # 1) Pulsação invisível
      if (In-PulseWindow $cfg.pulses.morning $tolerance) { Handle-Pulse -PulseName 'morning' -StateFile $stateFile -CareCfg $cfg.care }
      if (In-PulseWindow $cfg.pulses.midday  $tolerance) { Handle-Pulse -PulseName 'midday'  -StateFile $stateFile -CareCfg $cfg.care }
      if (In-PulseWindow $cfg.pulses.evening $tolerance) { Handle-Pulse -PulseName 'evening' -StateFile $stateFile -CareCfg $cfg.care }

      # 2) Ritmo da Entropia (escoamento leve)
      Soft-EntropyDrain -OverflowDir $env:__TEIA_OVERFLOW -BackupDir $env:__TEIA_BACKUP `
        -MaxCount $MaxOverflowCount -KeepDays $PurgeKeepDays -NoiseCfg $cfg.noise_vaccine

      # 3) Espelho contínuo e vacina (estado compacto)
      Write-Mirror "qol.heartbeat" @{
        overflow=(Get-ChildItem -LiteralPath $env:__TEIA_OVERFLOW -Force -ErrorAction SilentlyContinue | Measure-Object).Count
        backup  =(Get-ChildItem -LiteralPath $env:__TEIA_BACKUP -Force -ErrorAction SilentlyContinue | Measure-Object).Count
      }
    } catch {
      Write-Mirror "error.loop" @{ message=$_.Exception.Message }
    }

    if ($PSBoundParameters.ContainsKey('Once') -and $Once) { break }
    Start-Sleep -Seconds $CycleSeconds
  } while ($true)

  Write-Mirror "qol.stop" @{ reason = $(if($Once){"once"}else{"manual"}) }
}

# --- Ambiente de execução isolado ---
$env:__TEIA_ROOT     = $Root
$env:__TEIA_QOL      = $OpsDir
$env:__TEIA_OVERFLOW = $Overflow
$env:__TEIA_BACKUP   = $Backup

# Execução
Start-QoLLoop -CycleSeconds $CycleSeconds -MaxOverflowCount $MaxOverflowCount -PurgeKeepDays $PurgeKeepDays -Once:$Once
'@

# Grava o script operacional
Set-Content -Path $ScriptPath -Value $scriptContent -Encoding UTF8

# Saída mínima de confirmação
Write-Host "[OK] TEIA-Invisible-QoL.ps1 criado/atualizado em $ScriptPath"
Write-Host "Para executar (contínuo): pwsh -File `"$ScriptPath`""
Write-Host "Para ciclo único:         pwsh -File `"$ScriptPath`" -Once"
Write-Host "Logs/espelho:             $(Join-Path $OpsDir 'logs\mirror.jsonl')"
```

function In-PulseWindow($timeStr,[int]$toleranceMin){
  $now    = Get-Date

function Soft-EntropyDrain {
  param(
    [string]$OverflowDir,
    [string]$BackupDir,
    [int]$MaxCount,
    [int]$KeepDays,
    [object]$NoiseCfg
  )
  try {
    $items = Get-ChildItem -LiteralPath $OverflowDir -Force -ErrorAction SilentlyContinue
    if (-not $items) { return }

    # 1) Antirruído
    $filtered = foreach ($it in $items) {
      if ($it.PSIsContainer) { $it; continue }
      $ext = [IO.Path]::GetExtension($it.Name).ToLowerInvariant()
      if ($NoiseCfg.ignore_extensions -contains $ext) { continue }
      if ($it.Name.Length -gt $NoiseCfg.max_filename_length) { continue }
      $it
    }

    # 2) Purga suave por idade -> backup
    $threshold = (Get-Date).AddDays(-1 * $KeepDays)
    foreach ($it in $filtered) {
      if ($it.LastWriteTime -lt $threshold) {
        $dest = Join-Path $BackupDir $it.Name
        Move-Item -LiteralPath $it.FullName -Destination $dest -Force -ErrorAction SilentlyContinue
        Write-Mirror "entropy.purge_move" @{ name=$it.Name; to=$dest }
      }
    }

    # 3) Excesso acima do limite -> mover mais antigos
    $remaining = Get-ChildItem -LiteralPath $OverflowDir -Force -File -ErrorAction SilentlyContinue | Sort-Object LastWriteTime
    if ($remaining.Count -gt $MaxCount) {
      $excess = $remaining.Count - $MaxCount
      $toMove = $remaining | Select-Object -First $excess
      foreach ($it in $toMove) {
        $dest = Join-Path $BackupDir $it.Name
        Move-Item -LiteralPath $it.FullName -Destination $dest -Force -ErrorAction SilentlyContinue
        Write-Mirror "entropy.excess_move" @{ name=$it.Name; to=$dest }
      }
    }
  } catch {
    Write-Mirror "error.entropy" @{ message=$_.Exception.Message }
  }
}
