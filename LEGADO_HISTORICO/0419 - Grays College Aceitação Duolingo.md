- **Milímetros (mm):** É uma unidade pequena, o que pode tornar os números grandes e um pouco mais difíceis de trabalhar. Para um metro quadrado, você usaria 1000 mm tanto para largura quanto para altura.
- **Centímetros (cm):** É uma boa unidade de medida para trabalhos gráficos, especialmente em dimensões como um metro quadrado. Você usaria 100 cm para largura e altura.
- **Polegadas (in):** Comum no setor gráfico, especialmente em países que usam o sistema imperial. Para converter de centímetros para polegadas (1 in = 2,54 cm), você usaria aproximadamente 39,37 polegadas para largura e altura.
- **Pixels (px):** Usado principalmente para mídia digital, mas pode ser convertido para uso em impressões. A quantidade de pixels depende do DPI desejado.

### DPI:
Para uma impressão de alta qualidade, um DPI de 300 é padrão. Isso significa que para cada polegada da sua arte, haverá 300 pontos (ou pixels).
