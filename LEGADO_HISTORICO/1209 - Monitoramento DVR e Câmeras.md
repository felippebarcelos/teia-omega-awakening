### ASSISTENTE

Para integrar as imagens ao vivo do seu DVR Intelbras MHDX 1004 e das câmeras Jortan Wi-Fi 360° Yoosee em um computador Windows e visualizá-las através do aplicativo Alexa, siga os passos abaixo:

**1. Transmissão das Imagens do DVR Intelbras MHDX 1004 no Computador Windows:**

O DVR MHDX 1004 permite a transmissão ao vivo das câmeras conectadas através do protocolo RTMP. Para configurar essa funcionalidade:

- **Atualização de Firmware:** Certifique-se de que o firmware do seu DVR está atualizado para a versão que suporta RTMP. Você pode verificar e baixar a atualização no site oficial da Intelbras.

- **Configuração do RTMP:** No menu de configurações do DVR, habilite a função RTMP e configure os parâmetros necessários, como URL do servidor e chave de transmissão.

- **Software de Monitoramento:** Utilize o software Intelbras S.I.M. Play para monitorar as câmeras no seu computador Windows. Este software é compatível com o MHDX 1004 e permite a visualização ao vivo e gravação das imagens. citeturn0search14

Para um guia visual sobre como realizar transmissões ao vivo com o DVR MHDX, você pode assistir ao seguinte vídeo:

videoComo realizar transmissões ao vivo com DVRs MHDX Intelbrasturn0search1

**2. Integração das Câmeras Jortan Wi-Fi 360° Yoosee com o Computador Windows:**

As câmeras Jortan utilizam o aplicativo Yoosee para configuração e monitoramento. Para integrá-las ao seu computador:

- **Software Yoosee para PC:** Baixe e instale o software Yoosee compatível com Windows. Este software permite visualizar e gerenciar as câmeras diretamente do seu computador.

- **Configuração das Câmeras:** Siga as instruções do fabricante para conectar as câmeras à sua rede Wi-Fi e vinculá-las ao software Yoosee no PC.

Para um tutorial passo a passo sobre a configuração das câmeras Jortan Yoosee, assista ao vídeo abaixo:

**3. Visualização das Câmeras no Aplicativo Alexa:**

Atualmente, as câmeras que utilizam o aplicativo Yoosee, como as da marca Jortan, não possuem integração direta com o Amazon Alexa. Isso significa que não é possível visualizar as imagens dessas câmeras diretamente no aplicativo Alexa. citeturn0search5

**Considerações Finais:**

- **Monitoramento no Computador:** Você poderá monitorar tanto o DVR Intelbras quanto as câmeras Jortan no seu computador Windows utilizando os respectivos softwares de cada fabricante.

- **Integração com Alexa:** Para visualizar todas as câmeras no aplicativo Alexa, seria necessário utilizar dispositivos compatíveis com o Alexa, como câmeras que suportam a skill do Alexa. No entanto, as câmeras Jortan Yoosee não oferecem essa compatibilidade no momento.

Recomenda-se verificar regularmente as atualizações de firmware e software dos dispositivos, pois novas funcionalidades e compatibilidades podem ser adicionadas pelos fabricantes.

### ASSISTENTE

Para monitorar continuamente as imagens ao vivo do seu DVR Intelbras MHDX 1004 e das câmeras Jortan Wi-Fi 360° utilizando um computador com Windows, siga os passos abaixo:

**1. Monitoramento do DVR Intelbras MHDX 1004:**

O DVR MHDX 1004 permite acesso remoto às câmeras conectadas por meio de softwares compatíveis. A Intelbras oferece o software S.I.M. Play, que possibilita a visualização ao vivo e a reprodução de gravações.

- **Download e Instalação do S.I.M. Play:**
  - Acesse o site oficial da Intelbras e baixe o S.I.M. Play compatível com o seu sistema operacional Windows. citeturn0search14
  - Instale o software seguindo as instruções fornecidas.

- **Configuração do S.I.M. Play:**
  - Abra o S.I.M. Play e adicione o DVR utilizando o endereço IP local ou o domínio configurado para acesso remoto.
  - Insira as credenciais de acesso (usuário e senha) do DVR.
  - Após a configuração, será possível visualizar as câmeras conectadas ao DVR em tempo real.

**2. Monitoramento das Câmeras Jortan Wi-Fi 360°:**

As câmeras Jortan utilizam o aplicativo Yoosee para configuração e monitoramento. Para acessá-las via computador, é necessário utilizar o software CMS (Central Management System) compatível com o Yoosee.

- **Download e Instalação do CMS:**
  - Baixe o software CMS adequado para o Yoosee. citeturn0search1
  - Instale o software no seu computador seguindo as instruções fornecidas.

- **Configuração do CMS:**
  - Abra o CMS e faça login com as mesmas credenciais utilizadas no aplicativo Yoosee.
  - As câmeras previamente adicionadas no aplicativo serão sincronizadas automaticamente.
  - Caso contrário, adicione manualmente as câmeras inserindo o ID do dispositivo e a senha correspondente.

**Considerações Importantes:**

- **Conexão de Rede:**
  - Certifique-se de que o computador esteja conectado à mesma rede local que o DVR e as câmeras para facilitar a detecção e configuração dos dispositivos.

- **Manutenção do Computador:**
  - Para que o computador permaneça ligado continuamente, assegure-se de que ele esteja configurado para não entrar em modo de suspensão ou hibernação.
  - Considere utilizar um nobreak para proteger o equipamento contra quedas de energia.

- **Segurança:**
  - Mantenha os softwares e firmwares dos dispositivos atualizados para garantir a segurança do sistema.
  - Utilize senhas fortes e únicas para cada dispositivo e software de monitoramento.

Seguindo esses passos, você poderá monitorar de forma contínua e eficiente as imagens ao vivo do seu DVR Intelbras MHDX 1004 e das câmeras Jortan Wi-Fi 360° diretamente pelo seu computador com Windows.
