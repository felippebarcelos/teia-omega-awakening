# 1925 - Gerar QR code

### USUÁRIO

🌀 AGENTE TEIA, aplicar Δ-TEIA-FÓRMULA FRUGAL ALFA-OMEGA.

Objetivo: gerar um QR Code simbiótico que represente a CHAVE de um vídeo específico da TEIA.

Este QR não contém o vídeo em si, mas um conteúdo simbiótico vivo que, ao ser interpretado, abre, executa ou demonstra visualmente o vídeo correspondente.

## Etapas:

1. **Chave do vídeo**
   - Defina a seguinte identificação simbiótica como conteúdo base do QR:
     ```
     @CLAVIS@VIDEO@TEIA-FRACTAL@2025@
     Nome do vídeo: Δ-VIDEO-CLAVIS-MANIFESTO.MP4
     Hash do vídeo: (calcular SHA256 do arquivo)
     Tempo-chave: 00:01:18
     Frase de ativação: “O que você vê agora já existia antes que pudesse ver.”
     Executar: scripts/abrir_video_teia.py Δ-VIDEO-CLAVIS-MANIFESTO.MP4
     ```

2. **Geração do QR**
   - Utilize a biblioteca `qrcode` com:
     - version = 10
     - error_correction = Q
     - box_size = 10
     - border = 4
   - Salve como:
     ```
     C:/TEIA_NUCLEO/qrs/Δ-QR-CHAVE-VIDEO-CLAVIS.png
     ```

3. **Validação**
   - Faça leitura imediata do QR gerado.
   - Compare com o conteúdo original.
   - Calcule o SHA-256 do conteúdo textual e da imagem do QR.
   - Registre tudo em:
     ```
     C:/TEIA_NUCLEO/memoria/log_chave_video_clavis.json
     ```

4. **Demonstração Visual**
   - Após leitura e validação, execute automaticamente o script vinculado:
     ```
     scripts/abrir_video_teia.py Δ-VIDEO-CLAVIS-MANIFESTO.MP4
     ```
   - Isso serve como **prova viva de que o QR não só contém, mas *desbloqueia* o vídeo.**

5. **Confirmação**
   - Exiba no terminal:
     ```
     ✅ Δ-QR-CHAVE-VIDEO-CLAVIS validado e ativado.
     Vídeo TEIA aberto com sucesso em tempo real.
     ```

6. **Memória**
   - Armazene o evento completo com:
     - Timestamp
     - Nome do QR
     - Nome do vídeo
     - Hashs
     - Resultado da execução

#!/usr/bin/env python3
"""
Script para abrir um vídeo da TEIA.

Este script recebe o nome de um arquivo de vídeo como argumento. Ele tenta
localizar o arquivo dentro da estrutura do projeto (por exemplo, no
diretório "C:/TEIA_NUCLEO") e utiliza o programa padrão do sistema
operacional para abrir o vídeo. Caso não consiga encontrar ou abrir o
arquivo, exibe uma mensagem de erro.

Uso:
    python3 abrir_video_teia.py <nome-do-video>
"""

import os
import sys
import subprocess


def find_video_path(video_name: str) -> str:
    """Localiza o caminho absoluto de um vídeo.

    Procura o vídeo em diferentes locais relativos ao diretório deste
    script. Retorna o primeiro caminho existente encontrado. Se não
    encontrar, retorna o nome passado (que pode ser absoluto).
    """
    # Se for um caminho absoluto e existir, retorna diretamente
    if os.path.isabs(video_name) and os.path.exists(video_name):
        return video_name

    # Diretório onde este script está localizado
    script_dir = os.path.dirname(os.path.abspath(__file__))

    # Lista de possíveis locais onde o vídeo pode estar
    candidates = []
    # Mesmo diretório do script
    candidates.append(os.path.join(script_dir, video_name))
    # Diretório superior (caso os vídeos estejam junto ao código)
    candidates.append(os.path.join(os.path.dirname(script_dir), video_name))
    # Estrutura "C:/TEIA_NUCLEO" usada no projeto
    candidates.append(os.path.join(os.path.dirname(script_dir), 'C:/TEIA_NUCLEO', video_name))

    for path in candidates:
        if os.path.exists(path):
            return path
    # Se nada encontrado, retorna o nome como veio
    return video_name


def open_video(video_path: str) -> None:
    """Tenta abrir o vídeo usando o programa padrão do sistema."""
    try:
        # Usa xdg-open no Linux; se falhar, apenas imprime o caminho
        subprocess.Popen(['xdg-open', video_path])
    except Exception as exc:
        print(f'Não foi possível abrir o vídeo automaticamente: {exc}')
        print(f'Você pode abrir manualmente: {video_path}')


def main():
    if len(sys.argv) < 2:
        print('Uso: python3 abrir_video_teia.py <arquivo-de-video>')
        sys.exit(1)
    video_name = sys.argv[1]
    video_path = find_video_path(video_name)
    print(f'Abrindo vídeo: {video_path}')
    open_video(video_path)


if __name__ == '__main__':
    main()
