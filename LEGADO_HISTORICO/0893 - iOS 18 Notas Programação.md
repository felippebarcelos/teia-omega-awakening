# 0893 - iOS 18 Notas Programação

### USUÁRIO

É verdade que no iOS 18 o app notas vai poder ser usado como editor do código para programação 

---

### ASSISTENTE

Até o momento, não há informações oficiais sobre o app Notas no iOS 18 sendo atualizado para funcionar como um editor de código para programação. O app Notas da Apple é principalmente um aplicativo para anotações simples e textos, sem suporte avançado para recursos específicos de programação, como destaque de sintaxe, auto-completar, ou depuração.

Para programação em dispositivos iOS, existem outros aplicativos dedicados que oferecem ambientes mais adequados para desenvolvimento, como o Textastic, Pyto, o aplicativo Juno para Python, ou editores de código que suportam diversas linguagens, como o Code Editor.

Se precisar de mais informações ou tiver outra dúvida, fique à vontade para perguntar!
