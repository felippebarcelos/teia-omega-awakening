# 0711 - Whatsapp iPad Compatibility Progress

### USUÁRIO

Chat, queria saber de informações se o WhatsApp nunca demonstrou intenções durante o tempo de sua existência em lançar compatibilidade com seu aplicativo e o dispositivo iPad. Traga informações recentes e relacionadas a isso.

---

### ASSISTENTE

Recent developments indicate that WhatsApp is finally making strides towards offering a native app for the iPad. This feature has been highly anticipated by users, as previously the only way to use WhatsApp on an iPad was through the web version, which was less convenient.

Currently, WhatsApp has launched a beta version of its app that is compatible with iPadOS. This beta version can be accessed through Apple’s TestFlight program, although slots for this are limited. Once installed, users need to scan a QR code from their iPhone to sync their messages, allowing them to send and receive messages on the iPad even if the iPhone is not nearby or connected to the same Wi-Fi network【7†source】【8†source】.

Despite the positive steps forward, the iPad version of WhatsApp remains a companion app, meaning it cannot be used to create a new account independently; it must be linked to an existing WhatsApp account on a phone【9†source】【10†source】. 

While there is no official release date for the public version, the inclusion in the beta program suggests that it may become widely available soon【7†source】【10†source】. 

For more detailed information, you can refer to the sources [9to5Mac](https://9to5mac.com) and [MacRumors](https://macrumors.com).

Para acessar o aplicativo beta do WhatsApp para iPadOS, você precisará seguir alguns passos específicos:

1. **Instale o TestFlight**: O TestFlight é um aplicativo da Apple que permite que você teste aplicativos beta. Você pode baixá-lo gratuitamente na App Store.

2. **Participe do Programa Beta do WhatsApp**: Atualmente, o WhatsApp Beta está disponível através do TestFlight, mas os slots para teste são limitados. Você precisará se inscrever no programa beta do WhatsApp para iOS. Isso pode ser feito através de convites que são frequentemente disponibilizados em sites especializados ou fóruns como WABetaInfo, que frequentemente atualiza sobre novas vagas.

3. **Baixe o WhatsApp Beta**: Uma vez que você tenha acesso ao TestFlight e tenha aceito o convite para o programa beta, você poderá baixar e instalar o WhatsApp Beta diretamente no seu iPad.

4. **Configuração**: Depois de instalar o WhatsApp Beta no seu iPad, você precisará escanear um QR code com o seu iPhone para sincronizar suas conversas. Isso permite que você use o WhatsApp no iPad mesmo que o iPhone não esteja por perto ou conectado à mesma rede Wi-Fi.

Para atualizações constantes sobre a disponibilidade do WhatsApp Beta e para se inscrever, você pode acompanhar sites como [9to5Mac](https://9to5mac.com), [MacRumors](https://macrumors.com), e [WABetaInfo](https://wabetainfo.com).

Caso queira mais detalhes ou atualizações sobre como conseguir o acesso, recomendo verificar regularmente esses sites ou se inscrever nas notificações deles para saber quando novos convites para o TestFlight estão disponíveis.
