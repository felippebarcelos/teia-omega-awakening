# 0106 - Transferência de Windows para SSD

### USUÁRIO

Chat, antes eu tinha o meu Windows no disco C, onde era o HD externo de 1TB, onde era o local que o disco mantinha o Windows e inicializava. Porém, por eu obter o SSD de 120GB, pensei em instalar o Windows no SSD de 120GB e iniciar o Windows por ele e utilizar apenas o HD externo para armazenamento. Após instalar o SSD de 120GB, o Windows 11 não veio junto todos os arquivos, pré-definições, aplicativos e informações que estavam junto com o Windows dentro do disco C (atualmente A) e disco do ssd (C) atualmente. Como faço? Sabendo que estou com o meu Windows do zero e inicializei pelo SSD de 120GB para voltar a utilizar todas as informações, aplicativos e pré-definições utilizadas antigamente e que ainda estão dentro do HD externo do disco C.

---

### ASSISTENTE

Entendo sua situação. Você migrou o sistema operacional do Windows de um HD externo para um SSD, mas quer recuperar arquivos, configurações e aplicativos que estavam no HD antigo. Aqui estão algumas etapas que você pode seguir:

1. **Backup de Dados Importantes**: Antes de começar, faça backup de todos os dados importantes que estão no HD externo e no SSD.

2. **Transferência de Arquivos**: Você pode manualmente copiar seus arquivos pessoais (documentos, fotos, etc.) do HD externo para o SSD.

3. **Reinstalação de Aplicativos**: Infelizmente, aplicativos precisam ser reinstalados no novo sistema. Instale-os novamente no SSD.

4. **Transferência de Configurações**: Algumas configurações específicas do sistema e programas podem ser mais difíceis de transferir, pois podem depender de entradas específicas no registro do Windows. Para configurações de aplicativos específicos, veja se o aplicativo oferece uma opção de exportar/importar configurações.

5. **Utilização de Software de Clonagem**: Uma opção seria clonar o HD externo para o SSD. Isso criaria uma cópia exata do sistema operacional, configurações, aplicativos e arquivos no SSD. No entanto, dado que o SSD é menor, isso só seria viável se o uso de espaço no HD externo for menor que a capacidade do SSD.

6. **Ativação do Windows**: Se o Windows não está ativado no novo SSD, você precisará inserir a chave de produto para ativá-lo.

7. **Configuração do HD Externo para Armazenamento**: Depois de transferir tudo para o SSD, você pode usar o HD externo como armazenamento adicional. Certifique-se de formatá-lo adequadamente para evitar conflitos de inicialização.

Lembre-se, mudar o sistema operacional de um disco para outro pode ser um processo complexo e, às vezes, pode haver problemas de compatibilidade ou desempenho. Se você não se sentir confortável com esses processos, pode ser útil buscar ajuda profissional.
