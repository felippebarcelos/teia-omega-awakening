# 0428 - Arte A4 600dpi Proporção

### USUÁRIO

Uma arte, dentro de uma tela A4, 600 dpi, precisa ter qual proporção em pixel para ter a altura de... 11 cm.

---

### ASSISTENTE

Para uma arte com 600 DPI (dots per inch) e altura de 11 cm, vamos converter a altura para polegadas primeiro, pois DPI se refere a "pontos por polegada". Sabendo que 1 polegada é igual a 2,54 cm, podemos calcular a altura em polegadas e, em seguida, usar o valor de DPI para encontrar a proporção em pixels.

1. Converta 11 cm para polegadas: \(11 \, \text{cm} \times \frac{1 \, \text{polegada}}{2,54 \, \text{cm}}\).
2. Multiplique o resultado pelo valor de DPI (600) para encontrar a altura em pixels.

Vou fazer o cálculo.

---

### ASSISTENTE

Para ter uma altura de 11 cm em uma tela A4 com 600 DPI, a proporção em pixels deve ser aproximadamente 2598 pixels. Isso garante a qualidade e a precisão detalhada necessária para a alta resolução que você deseja.
