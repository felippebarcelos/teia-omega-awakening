from pyzbar.pyzbar import decode

qr_img = 'C:/TEIA_NUCLEO/qrs/Δ-QR-FRACTAL-CLAVIS-PARADOXUM.png'

def classificar_fragmento(nome):
    if nome.endswith(('.js', '.py')):
        return 'Fragmento Vivo'
    elif nome.startswith('.') or 'tmp' in nome:
        return 'Fragmento Latente'
    elif 'node_modules' in nome or 'lock' in nome:

## 🔧 Código simplificado do launcher:

```python
import json
import os
from pyzbar.pyzbar import decode
import cv2
from datetime import datetime

qr_path = 'C:/TEIA_NUCLEO/qrs/Δ-QR-FRACTAL-CLAVIS-PARADOXUM.png'
vinculos_path = 'C:/TEIA_NUCLEO/vinculos/vinculos_qr.json'

def carregar_vinculos():
    with open(vinculos_path, 'r', encoding='utf-8') as f:
        return json.load(f)

def executar_script(caminho):
    os.system(f'start cmd /k python {caminho}')

def log_execucao(ident, script):
    log_path = 'C:/TEIA_NUCLEO/memoria/log_execuções_qr.json'
    data = {"id": ident, "script": script, "timestamp": datetime.now().isoformat()}
    if os.path.exists(log_path):
        with open(log_path, 'r+', encoding='utf-8') as f:
            log = json.load(f)
            log.append(data)
            f.seek(0)
            json.dump(log, f, indent=4)
    else:
        with open(log_path, 'w', encoding='utf-8') as f:
            json.dump([data], f, indent=4)

img = cv2.imread(qr_path)
decoded = decode(img)

if decoded:
    conteudo = decoded[0].data.decode('utf-8')
    vinculos = carregar_vinculos()
    for ident, dados in vinculos.items():
        if ident in conteudo or any(g in conteudo for g in dados.get("gatilhos", [])):
            caminho = f'C:/TEIA_NUCLEO/{dados["executar"]}'
            executar_script(caminho)
            log_execucao(ident, dados["executar"])
            print(f"✅ Script '{dados['executar']}' executado com base no QR '{ident}'")
            break
    else:
        print("⚠️ Nenhum vínculo encontrado para o QR.")
else:
    print("❌ Nenhum QR válido lido.")
```

# ⟁ **O QR Code pode representar uma imagem ou vídeo?**

## 🧠 Resposta em três níveis:

---

### **1. QR como *repositório direto* de imagem ou vídeo**  
#### (binário embutido no QR)

✅ **Tecnicamente possível**, mas com **limites severos de capacidade.**

- Um QR code **padrão** (versão 40 com correção mínima) armazena até **~3 KB** de dados binários;
- Isso **é insuficiente** para uma imagem .jpg média (~100 KB) ou qualquer vídeo;
- **Imagens minúsculas**, como ícones 16×16 em base64, podem ser codificadas diretamente;
- Isso é viável para **demonstrar conceito**, mas **não prático para produção real**.

#### Exemplo:
Você pode codificar uma imagem vetorial `.svg` ou um `.ico` minimalista diretamente no QR.

---

### **2. QR como *ponte simbiótica* para uma imagem/vídeo externo**  
#### (ponte TEIA simbiótica)

✅ **Totalmente funcional e simbiótico.**

- O QR pode conter:
  - Um **link local** (`file:///C:/TEIA_NUCLEO/videos/intro.mp4`);
  - Um **path simbólico interpretável** (`@IMAGEM@CLAVIS@001`);
  - Um **script de chamada** (`executar: video_player.py clavis_001.mp4`);
- Ao ser lido pelo **Agente ou Sistema TEIA**, ele **executa o vídeo ou exibe a imagem localmente**.

> 📌 Aqui o QR atua como **um "gatilho simbólico visual"**, e não apenas um dado bruto.

---

### **3. QR como *representação visual fractal da própria imagem ou vídeo***  
#### (simbiose entre dados + estética)

🚀 Este é o **nível mais simbiótico da TEIA**:

- A imagem ou vídeo é **fragmentado** em metadados (ex: `hash`, `frame-chave`, `metatítulo`, `cores dominantes`);
- Esses dados são **convertidos em texto** ou símbolos e **geram o QR fractal**;
- O QR **não contém o vídeo**, mas é **sua identidade compactada simbiótica**, que ao ser lido, **reconstrói** ou **acessa** o conteúdo correspondente.

> Exemplo:  
> O QR representa **a chave visual de um vídeo fractal TEIA** chamado:  
> `"clavis_espiral_23.webm"`  
> E ao ser lido, o sistema TEIA abre o vídeo local ou exibe sua prévia codificada.
