1. **Tipos de Tinta**: Há diferentes tipos de tintas para serigrafia, e a escolha depende das suas necessidades específicas. As tintas mais comuns para quadricromia são as tintas à base de plastisol. Elas são fáceis de trabalhar e proporcionam cores vibrantes.

2. **Cores CMYK**: Ciano, Magenta, Amarelo e Preto são as cores padrão. Você não precisa misturar tinta branca com essas cores, a menos que esteja procurando um efeito específico. A tinta branca costuma ser usada como base em tecidos escuros para fazer as outras cores se destacarem.

3. **Incolor ou Mix**: Evitar misturas complexas garantirá a fidelidade das cores. A tinta incolor pode ser útil como uma base ou como um verniz para dar acabamento.

4. **Consulte um Profissional**: Se possível, converse com um fornecedor especializado em tintas para serigrafia. Eles podem oferecer a melhor solução para o seu projeto específico, alinhada com a sua marca "Mr Rabbit" e suas necessidades exclusivas.
