# 0548 - Cascalho para cannabis.

### USUÁRIO

cascalho é bom para criar uma base para o pote de plantação da cannabis?

---

### ASSISTENTE

Cascalho pode ser uma boa opção para criar uma base em potes de plantação de cannabis, especialmente se você deseja melhorar a drenagem. Ele é eficaz em evitar que a água fique estagnada nas raízes, o que é crucial para a saúde das plantas de cannabis, que não toleram "pés molhados". No entanto, é importante notar que o cascalho sozinho não fornecerá os nutrientes necessários para o crescimento ótimo da planta. É recomendável usar o cascalho como uma camada de base e preencher o resto do pote com um substrato apropriado que contenha uma mistura de terra, perlita, vermiculita e compostos orgânicos, que fornecerão suporte nutricional e retenção de água adequada.
