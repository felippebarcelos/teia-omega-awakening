### ASSISTENTE

Entendo que você deseja instalar o Agent DVR no seu computador e está considerando as opções de download disponíveis no site do iSpy. Vamos esclarecer suas dúvidas:

**1. Seleção da Versão Adequada para o Seu Computador:**

Para instalar o Agent DVR no seu computador, é fundamental escolher a versão compatível com o seu sistema operacional. As opções disponíveis são:

- **Agent DVR Windows:** Destinado a computadores com Windows 64 bits.
- **Agent DVR 32-bit Windows:** Para sistemas Windows de 32 bits.
- **Agent DVR MacOS:** Para computadores Apple com macOS.
- **Agent DVR Linux:** Para sistemas operacionais baseados em Linux.
- **Agent DVR RPI:** Específico para Raspberry Pi.
- **Agent DVR Docker:** Para instalação em ambientes Docker.

Se o seu computador utiliza Windows 64 bits, a opção recomendada é o **Agent DVR Windows**. Caso seu sistema seja Windows 32 bits, selecione o **Agent DVR 32-bit Windows**. Para verificar a versão do seu Windows, acesse as configurações do sistema ou o Painel de Controle.

**2. Possibilidade de Instalar o Agent DVR Diretamente no DVR Intelbras:**

Os DVRs da Intelbras possuem sistemas operacionais proprietários, projetados especificamente para funções de gravação e monitoramento de vídeo. Esses sistemas não permitem a instalação de softwares de terceiros, como o Agent DVR. Portanto, não é possível instalar o Agent DVR diretamente no seu DVR Intelbras.

**3. Integração do DVR Intelbras com o Agent DVR no Computador:**

Embora não seja viável instalar o Agent DVR no DVR Intelbras, é possível integrar as câmeras conectadas ao seu DVR com o Agent DVR instalado no seu computador. Para isso, siga os seguintes passos:

- **Obtenha as Informações de Acesso ao DVR:** Anote o endereço IP, porta, nome de usuário e senha do seu DVR Intelbras.

- **Adicione o DVR ao Agent DVR:** No Agent DVR, utilize a função de adicionar dispositivo e selecione a opção de DVR. Insira as informações coletadas (endereço IP, porta, nome de usuário e senha) para estabelecer a conexão. Certifique-se de que o Agent DVR seja compatível com o modelo do seu DVR Intelbras. citeturn0search6

- **Configure as Câmeras:** Após a conexão bem-sucedida, você poderá visualizar e gerenciar as câmeras conectadas ao seu DVR através do Agent DVR no seu computador.

Essa integração permite que você utilize as funcionalidades avançadas do Agent DVR, como detecção de movimento e gravação programada, aplicadas às câmeras do seu sistema Intelbras.
