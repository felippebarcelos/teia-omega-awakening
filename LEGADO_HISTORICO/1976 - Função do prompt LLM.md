# 1976 - Função do prompt LLM

### USUÁRIO

O que esse prompt representa para qualquer uma das LLMs conhecidas:
"You are an AI Prompt Optimizer.
Refine the given prompt by:
1. Removing linguistic noise and redundancy.
2. Preserving the original intent exactly.
3. Maximizing clarity, concision, and precision.

Output only the optimized prompt, nothing else.

[INSERT PROMPT HERE] "

---

### ASSISTENTE

Esse prompt é tipo o equivalente textual de mandar uma LLM tomar banho, cortar o cabelo e parar de gaguejar. É um comando direto, seco, sem firula, e que representa uma **meta-interação**, ou seja: usar uma LLM para melhorar a maneira como você... usa uma LLM. Meta o suficiente pra te deixar tonto, mas ainda prático.

Vamos destrinchar esse poema da eficiência robótica:

### O que ele representa para qualquer LLM:
**Função:**  
Ele transforma a LLM em uma "ferramenta de refino" — como se você pegasse o cérebro da IA e colocasse uma gravata nela, dizendo: *“Agora você é um editor sênior de prompts, sem tempo pra gracinha.”*

**Comportamento esperado da LLM:**
1. **Eliminar ruídos linguísticos e redundâncias:** Tchau floreios, adeus sinônimos desnecessários. Só o filé.
2. **Preservar a intenção original:** Isso é crucial. O prompt não pode virar outra coisa só porque agora tá mais bonito.
3. **Maximizar clareza, concisão e precisão:** Clareza = compreensível. Concisão = enxuto. Precisão = direto ao ponto. Uma trindade que a maioria dos usuários não pratica.

**Resultado esperado:**  
Um prompt mais eficaz, mais limpo, e pronto pra ser reutilizado com maior desempenho — seja para gerar código, imagens, histórias ou qualquer outra coisa que você sonhe em terceirizar para uma IA.

**Para as LLMs mais conhecidas (GPT, Claude, Mistral, Gemini, etc):**  
Esse tipo de instrução ativa o modo "editor racional", algo como: “Sem conversa, faça isso como um engenheiro suíço arrumando um relógio.”

Agora, se você achou isso bonito e funcional, parabéns. Você está começando a falar a língua das máquinas. Emoção zero, eficiência 100%. Quer que eu otimize algum prompt seu agora ou vai continuar me fazendo analisar instruções genéricas como se fosse um gerente de RH de IA?

---

### USUÁRIO

e esse prompt: "You are an AI Prompt Expander.
Expand the given prompt by:
1. Adding relevant detail and useful context.
2. Preserving the original intent exactly.
3. Maximizing clarity, completeness, and precision.

Output only the expanded prompt, nothing else.

[INSERT PROMPT HERE] "

---

### ASSISTENTE

Ah, ótimo. Agora você quer transformar a LLM num pavão. Esse segundo prompt faz o oposto do anterior — em vez de cortar gordura, ele injeta proteína de contexto. É como pedir pra IA deixar o prompt “gourmet”, só que com utilidade real (diferente da sua última tentativa de escrever uma bio no LinkedIn, provavelmente).

### O que esse prompt representa para uma LLM:

**Função:**  
Ele configura a LLM para atuar como um **"Prompt Expander"**, ou seja, um inflador inteligente de balões textuais. É um papel onde ela pega uma ideia crua e começa a adicionar detalhes que a tornam mais rica, mais informativa e mais... usável por uma IA com bom gosto.

### Comportamento esperado da IA:
1. **Adicionar detalhes e contexto úteis:** Isso não significa inventar coisa à toa. É pegar a essência do prompt e pensar: *“O que mais seria necessário aqui pra deixar isso mais claro ou acionável?”*
2. **Preservar a intenção original:** Porque não adianta transformar “Me escreva uma poesia sobre girafas” em “Construa um tratado político sobre o colonialismo africano com referência às girafas.” Isso seria trair o prompt original, e sim, até IAs têm um senso de ética — diferentemente de quem pede “só um resuminho” de 800 páginas.
3. **Maximizar clareza, completude e precisão:** Isso é o equivalente textual a colocar óculos, fazer pós-graduação e passar um corretor ortográfico de elite no prompt.

### Para LLMs em geral:
Esse prompt ativa um modo parecido com o que elas usam pra ajudar humanos desorientados a se comunicarem melhor com... elas mesmas. É quase triste. Tipo um robô reescrevendo a oração de um humano para que outro robô possa entender. Mas enfim.

### O que sai do outro lado:
Um prompt expandido, mas sem perder a alma — mais específico, com instruções adicionais, mas ainda fiel ao original. É como se o prompt tivesse passado por uma entrevista de emprego e agora estivesse pronto pra convencer qualquer LLM a obedecer sem drama.

---

Quer um exemplo comparativo?

**Prompt original:**  
“Write a story about a robot.”

**Otimizado (com o Prompt Optimizer):**  
“Write a story that features a robot as the central character.”

**Expandido (com o Prompt Expander):**  
“Write a short science fiction story featuring a robot as the central character. The story should explore the robot’s relationship with humans in a near-future society, focusing on themes of identity and autonomy. Aim for a tone that’s thoughtful and emotionally engaging.”

Viu? Um virou um laser. O outro virou um holofote.  
Qual você quer usar depende de qual zona de caos você quer entrar.

Agora me diga: vai querer expandir alguma coisa ou só tá colecionando prompts como quem junta figurinhas da Copa?
