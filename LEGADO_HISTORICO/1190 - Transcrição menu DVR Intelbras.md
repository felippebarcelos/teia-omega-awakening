### Resumo Completo de Todas as Imagens do DVR Intelbras MHDX 1004

#### Menu Principal do Sistema
O menu do DVR Intelbras MHDX 1004 é organizado em várias seções principais: **Câmera**, **Rede**, **Evento**, **Armazenamento** e **Sistema**. Cada seção possui opções de configuração detalhadas, permitindo a personalização e o controle completo das funcionalidades do dispositivo.

#### 1. **Câmera**
   - **Status dos Canais**: Exibe cada canal configurado com formato de vídeo (720P) e modo de operação (CVI).
   - **Encoder**: Configura o tipo, compressão, resolução, taxa de quadros e outras definições de codificação de cada canal.
   - **Nome do Canal**: Permite atribuir nomes a cada canal, facilitando a identificação.
   - **Modo de Operação**: Define o tipo de entrada para cada canal, incluindo opções como AHD, CVBS, TVI e IP.
   - **Disp. Remoto, Imagem e Atualizar Câmera**: Outras configurações para gerenciar dispositivos de câmeras e ajustar qualidade de imagem.

#### 2. **Rede**
   - **Intelbras Cloud**: Habilita o recurso Intelbras Cloud, permitindo monitoramento remoto. Exibe ID do dispositivo e QR code para conexão.
   - **TCP/IP**: Configura o IP, máscara de sub-rede, gateway e DNS.
   - **Portas**: Define as portas de serviço, incluindo HTTP, HTTPS, TCP, UDP e RTSP.
   - **DDNS**: Configurações para usar um serviço de DNS dinâmico (Intelbras DDNS).
   - **Filtro IP**: Permite habilitar um filtro de IP para restringir acesso.
   - **UPnP**: Habilita UPnP para facilitar o redirecionamento de portas.
   - **SNMP**: Configuração para monitoramento de rede usando SNMP.
   - **802.1x**: Configura segurança de rede através de 802.1x com opções de autenticação EAP-MD5.

#### 3. **Sistema**
   - **Geral**: Configurações principais, incluindo idioma, padrão de vídeo, visualização em tempo real e opções de auto-saída.
   - **Tela e PTZ**: Ajustes de tela e controle PTZ, como protocolo e baudrate.
   - **Multi Box**: Gerenciamento do Multi Box (caso esteja conectado).
   - **Conta**:
     - **Usuário**: Gestão de contas de usuários com permissões de administrador e usuário.
     - **Grupo**: Definições de grupos de permissões (administrador e usuário).
     - **Usuário ONVIF**: Gerenciamento de contas ONVIF para compatibilidade com dispositivos de terceiros.
   - **Automanut.**: Opções de manutenção automática do dispositivo.
   - **Configuração Backup**: Opções para backup das configurações.
   - **Padrão**: Restaura configurações de fábrica ou ajusta configurações específicas para manter como padrão.
   - **Atualização**: Permite atualizações manuais via USB e online, com verificação automática de novas versões.
