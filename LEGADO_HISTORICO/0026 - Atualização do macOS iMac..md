# 0026 - Atualização do macOS iMac.

### USUÁRIO

Restaurei meu imac para ajustes de fabrica e voltou para o mac os x mavericks, onde nao consigo instalar o opencore patcher porque solicita a versao minima do mac os Yosenite, como faço para atualizar meu imac later 2013?

---

### ASSISTENTE

Situação interessante! Vamos direto ao ponto:

1. **Atualização Direta**: Abra a Mac App Store e tente fazer a atualização diretamente para a versão mais recente do macOS que seu iMac suporta.
  
2. **USB Bootable**: Crie um drive USB inicializável com o macOS que você deseja instalar. Inicialize o iMac a partir deste drive e instale o novo macOS.

3. **Outro Mac**: Se você tiver acesso a outro Mac, pode baixar a versão desejada do macOS e usar o modo de recuperação para instalar no seu iMac 2013.

4. **Hack Esperto**: Se o OpenCore Patcher não funciona porque você precisa do Yosemite, que tal criar um ambiente virtual do macOS Yosemite em seu iMac e executar o patcher lá? Não é o método padrão, mas você é um criativo, certo?

Lembre-se de fazer backup de todos os seus dados importantes antes de fazer qualquer atualização importante do sistema.

### ASSISTENTE

O iMac Late 2013 suporta até o macOS Monterey, que é a versão 12 do sistema operacional. Contudo, informações atualizadas podem mostrar uma versão mais recente que ele suporte, então vale a pena verificar.

1. Abra a Mac App Store.
2. Vá na aba "Atualizações" ou procure pela versão do macOS que deseja instalar na barra de pesquisa.
3. Siga as instruções para baixar e instalar.

Rápido e fácil! E, claro, sempre faça backup antes de qualquer atualização. Como você é alguém que está acostumado a solucionar problemas de forma criativa, fique atento aos detalhes durante o processo para evitar contratempos.
