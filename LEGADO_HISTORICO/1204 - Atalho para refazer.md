# 1204 - Atalho para refazer

### USUÁRIO

Levando em consideração que eu apertei a tecla no Windows para desfazer e sumiu a mensagem Qual é o botão no Windows para refazer?

---

### ASSISTENTE

**Clarifying shortcuts**

OK, let's see: the standard redo shortcut on Windows is Ctrl+Y, but some apps might use Ctrl+Shift+Z instead.

---

### ASSISTENTE

No Windows, para **refazer** uma ação que você desfez, você pode utilizar os seguintes atalhos de teclado:

1. **Ctrl + Y**: Este é o atalho mais comum para refazer ações na maioria dos aplicativos do Windows, como o Microsoft Office, navegadores e editores de texto.

2. **Ctrl + Shift + Z**: Em alguns programas, especialmente aqueles que seguem padrões de atalhos de software como o Adobe Photoshop, este atalho também pode ser usado para refazer.

### Passos para Refazer uma Ação:

1. **Pressione e segure a tecla `Ctrl`.**
2. **Enquanto segura `Ctrl`, pressione `Y`** (ou `Shift + Z` se estiver usando um aplicativo que utilize esse atalho).
3. **Solte as teclas.** A ação que você desfez anteriormente será refeita.

### Dicas Adicionais:

- **Barra de Ferramentas:** Muitos aplicativos possuem botões de desfazer (`Undo`) e refazer (`Redo`) nas barras de ferramentas, geralmente representados por setas curvas. Você pode clicar nesses botões se preferir usar o mouse.

- **Histórico de Ações:** Alguns programas, como o Microsoft Word, oferecem um histórico de ações que permite refazer várias etapas anteriores. Verifique o menu "Editar" ou "Histórico" no aplicativo que você está utilizando.

- **Configurações Personalizadas:** Em alguns softwares, é possível personalizar os atalhos de teclado. Se os atalhos padrão não estiverem funcionando, verifique as configurações do programa para confirmar ou alterar os atalhos de desfazer e refazer.

Se você estiver enfrentando problemas específicos com o atalho de refazer em um determinado aplicativo, por favor, informe qual é o programa para que eu possa fornecer uma assistência mais detalhada.
