# 0911 - Extensão Chrome para Transcrição

### USUÁRIO

Qual é a extensão do chrome que permite transcrição no chatgpt?

---

### ASSISTENTE

Para transcrever suas interações com o ChatGPT no navegador Chrome, você pode utilizar a extensão **"ChatGPT to Markdown"**. Essa extensão permite que você salve o conteúdo das conversas diretamente em formato Markdown, facilitando a transcrição e o armazenamento do texto.

Outra opção é usar extensões como **"Speech to Text"**, como o **"Voice Control for ChatGPT"**. Essa extensão permite que você fale diretamente ao ChatGPT e o texto transcrito da sua fala seja enviado como entrada de texto. No entanto, essa ferramenta não transcreve o conteúdo gerado pelo ChatGPT; apenas transforma sua fala em texto para ser usado na interação.

Se o objetivo for transcrever automaticamente todo o conteúdo da conversa, o uso de ferramentas adicionais, como scripts personalizados ou extensões específicas para captura de tela e texto, pode ser necessário.

Para instalar essas extensões, você pode procurar por elas na Chrome Web Store.
