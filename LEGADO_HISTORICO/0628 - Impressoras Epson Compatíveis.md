# 0628 - Impressoras Epson Compatíveis

### USUÁRIO

Quais são as impressoras compatíveis com Epson Print Layout? Aplicativo

---

### ASSISTENTE

O aplicativo Epson Print Layout é compatível com uma variedade de impressoras Epson, especialmente aquelas voltadas para impressão de fotos de alta qualidade e impressão de grande formato. As impressoras compatíveis incluem modelos das séries SureColor e EcoTank, entre outros. Aqui estão alguns dos modelos principais:

1. **SureColor Série P**: 
   - P400
   - P600
   - P700
   - P800
   - P900
   - P5000
   - P6000
   - P7000
   - P7570
   - P8000
   - P9000
   - P9570

2. **EcoTank Série ET**:
   - ET-8500
   - ET-8550

3. **Modelos adicionais**:
   - SC-P700
   - SC-P900
   - L8160
   - L8180

Essas impressoras são projetadas para maximizar a qualidade e flexibilidade na impressão, permitindo ajustes detalhados de layout e gerenciamento de cores através do Epson Print Layout【7†source】【8†source】【9†source】【10†source】.

### USUÁRIO

L3250 não é suportada?

---

### ASSISTENTE

A impressora Epson EcoTank L3250 não está listada entre as impressoras compatíveis com o aplicativo Epson Print Layout. O aplicativo Epson Print Layout é geralmente destinado a modelos de impressoras de grande formato e fotográficas que suportam ajustes avançados de layout e gerenciamento de cores. A L3250, sendo uma impressora multifuncional da linha EcoTank, não possui os recursos avançados necessários para ser compatível com o Epson Print Layout.

Se você precisa de uma solução para impressão de alta qualidade ou gerenciamento de layouts detalhados, pode considerar um dos modelos mencionados anteriormente, como aqueles da série SureColor ou EcoTank ET-8500 e ET-8550, que são compatíveis com o Epson Print Layout【7†source】【8†source】【10†source】.
