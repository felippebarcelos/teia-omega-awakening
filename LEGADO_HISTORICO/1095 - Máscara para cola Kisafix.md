# 1095 - Máscara para cola Kisafix

### USUÁRIO

qual é a máscara mais ideal pra aplicar cola de condado kisafix

---

### ASSISTENTE

Para aplicar cola de contato, como a Kisafix, é importante utilizar uma máscara adequada para proteger-se dos vapores tóxicos que podem ser liberados durante o processo. A máscara ideal seria uma máscara com filtro para vapores orgânicos, que é especificamente projetada para proteger contra produtos químicos voláteis.

A recomendação seria uma **máscara com filtros P2 ou P3 e cartucho para vapores orgânicos (filtro VO/GA)**, que protegem contra a inalação de vapores de solventes presentes na cola. Máscaras respiratórias como as da 3M, modelos da série **6000 ou 7000 com filtros apropriados**, são boas opções. Além disso, é importante garantir que o ambiente seja bem ventilado durante a aplicação da cola.
