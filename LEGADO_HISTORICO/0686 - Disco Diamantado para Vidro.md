# 0686 - Disco Diamantado para Vidro

### USUÁRIO

“DISCO DIAMANTADO PARA CERÂMICA 230 X 22.23MM 10MM 2608602205 BOSCH” esse disco inserido em uma esmilhadeira consegue cortar tubo de vidro de 6mm de forma precisa sem danificar o resto da estrutura desse tubo de vidro comprado em casa de produtos para laboratórios?

---

### ASSISTENTE

Um disco diamantado para cerâmica, como o "DISCO DIAMANTADO PARA CERÂMICA 230 X 22.23MM 10MM 2608602205 BOSCH", é projetado principalmente para cortar materiais duros e quebradiços como cerâmica e porcelanato. No entanto, o corte de vidro, especialmente vidro de 6mm, requer algumas considerações específicas para evitar danos ao material.

### Considerações ao Cortar Vidro com Disco Diamantado

1. **Tipo de Disco**: Embora discos diamantados sejam resistentes, eles são otimizados para materiais como cerâmica. Para vidro, um disco diamantado específico para vidro seria mais adequado, pois tem uma borda mais suave que minimiza a chance de lascar.

2. **Velocidade de Corte**: A velocidade da esmilhadeira deve ser ajustada para evitar sobreaquecimento e rachaduras no vidro. Uma velocidade mais lenta e controlada é preferível.

3. **Refrigeração**: Usar água ou outro refrigerante durante o corte é essencial para evitar o superaquecimento do vidro e do disco, o que pode levar a trincas e lascas.

4. **Pressão Aplicada**: Deve-se aplicar uma pressão leve e constante para evitar que o vidro se quebre devido a forças excessivas.

5. **Preparação do Vidro**: Marcar a linha de corte e, se possível, fazer um corte inicial superficial pode ajudar a guiar o disco e minimizar danos.

### Recomendação

Apesar de ser possível usar o disco diamantado mencionado, a precisão e a integridade do tubo de vidro podem não ser ideais. Discos diamantados específicos para vidro, disponíveis em lojas especializadas, oferecem melhores resultados para cortes precisos sem danificar a estrutura do vidro.

Se o corte de vidro for uma tarefa frequente, investir em um disco adequado para vidro é a melhor opção. Se for uma tarefa única, é crucial tomar todas as precauções mencionadas para minimizar o risco de dano.

**Conclusão**: Usar o "DISCO DIAMANTADO PARA CERÂMICA 230 X 22.23MM 10MM 2608602205 BOSCH" pode funcionar, mas com o risco de danificar o tubo de vidro. Considerar um disco específico para vidro e seguir as precauções detalhadas aumentará significativamente as chances de um corte bem-sucedido e preciso.
